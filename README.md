# PaisaCalc

## Overview

PaisaCalc is a modern personal finance platform designed to help Indian users make informed financial decisions through easy-to-use calculators and financial planning tools.

The application provides a collection of calculators and utilities for evaluating loans, investments, taxes, savings, and other financial scenarios. It aims to simplify complex financial calculations and present results in a clear and user-friendly manner.

Built using modern web technologies and enhanced with AI-assisted development workflows, PaisaCalc focuses on accessibility, accuracy, and ease of use.

## Installation

### Clone Repository

```bash
git clone https://github.com/Ajay190504/PaisaCalc.git
cd PaisaCalc
```

### Install Dependencies

```bash
npm install
```

### Environment Setup

Create a `.env.local` file in the project root:

```env
GEMINI_API_KEY=your_gemini_api_key
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
